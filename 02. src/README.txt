Path Planning with Dijkstra's Algorithm
--------------------------------------------------------------------------------
Dependencies
--------------------------------------------------------------------------------
To run this project, you need the following dependencies:

Python 3.x
OpenCV (cv2)
numpy

--------------------------------------------------------------------------------
Running the Code
--------------------------------------------------------------------------------
Clone or download the repository to your local machine.

```
https://github.com/hoangpm99/enpm-661-project-2-dijkstras-path-planning.git
```

Ensure you have Python and OpenCV installed.
Navigate to the project directory in your terminal or command prompt.
Run the following command to execute the program:

python dijkstra_hoang_pham.py

You will be prompted to enter the start and goal coordinates for the path planning. Enter the coordinates in the format x y and press Enter.

Another option is open file in VSCode and use CodeRunner.
